## 전체_피드보기

- 유스케이스명: 전체 피드보기
- 액터 : 비회원,일반회원,버스커,제공자,관리자
- 시작 조건 : 없음
- 종료 조건 : 인기버스커에 따른 버스커목록이 출력된다. 
- 성공 시나리오
	- 사용자 : 상단의 피드보기 버튼을 클릭한다
	- 시스템 : 인기버스커(좋아요 순위)로 버스커목록이 출력된다. 
- 예외 시나리오

- 시작 조건 : 없음

- 예외 시나리오 : 검색한 이름의 버스커가없으면 얼라 창을 띄운다
